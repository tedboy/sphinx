sphinx.pycode.AttrDocVisitor¶

class sphinx.pycode.AttrDocVisitor(number2name, *args)[source]¶

Visitor that collects docstrings for attribute assignments on toplevel and in classes (class attributes and attributes set in __init__).

The docstrings can either be in special ‘#:’ comments before the assignment or in a docstring after it.
