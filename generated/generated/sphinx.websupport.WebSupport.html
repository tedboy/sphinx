7.1.2.6. sphinx.websupport.WebSupport¶

class sphinx.websupport.WebSupport(srcdir=None, builddir='', datadir=None, staticdir=None, doctreedir=None, search=None, storage=None, status=<open file '<stdout>', mode 'w'>, warning=<open file '<stderr>', mode 'w'>, moderation_callback=None, allow_anonymous_comments=True, docroot='', staticroot='static')[source]¶

The main API class for the web support package. All interactions with the web support package should occur through this class.
